# alick 毕业快乐趣味交互网站设计方案

本项目旨在为 Alick 制作一个可爱俏皮的、富有北大和元培特色、融合恋爱纪念和王者荣耀彩蛋的移动端单页面交互网站。

## 用户审核要点

- **运行平台**：优化自适应手机浏览器（特别是微信内置浏览器）。
- **图片素材**：已提供 5 张现场照片。将放置在 `assets/` 文件夹下并在页面中进行展示与轮播。
- **背景音乐**：提供轻快的背景音乐（带静音切换开关）。

## 设计实现详情

### 1. 结构设计 (HTML5)
整个网页采用单页式全屏滑动设计，分为以下三个主要状态卡片，卡片切换带磨砂玻璃模糊和淡入淡出动画：

- **Section 1: 树洞相遇 (Met-Cute)**
  - 仿照北大微助手树洞样式，呈现 `#102400 (洞主) 男好想谈恋爱` 与 `a女好想谈恋爱`。
  - 点击“开启魔法”按钮后，执行转场。
- **Section 2: 毕业冲刺小游戏 (Honor of Kings game)**
  - 核心互动：限定时间内快速点击“水晶”，每次点击浮现 “绩点 +10” 或 “大融城美食 +1”。
  - 连击特效：1-3-5-8-12 击分别发出王者荣耀 First Blood 至 Penta Kill 语音（用 Web Audio API 播放音效，或显示华丽特效字）。
  - 通关奖励：弹出 Victory 标志，并获得“荣耀毕业生 · 大融城特级美食家”奖牌。
- **Section 3: 拨穗与扔学士帽 (Turning Tassel & Confetti)**
  - 界面呈现：一顶插画风格的学士帽，流苏在右侧。
  - 手势动作：手指水平左滑流苏，流苏随手指平滑移动到左侧。
  - 抛帽彩蛋：点击学士帽，学士帽缩放飞天，触发漫天炫彩纸屑（canvas-confetti）。
  - 终极展示：照片轮播展示（使用用户提供的 5 张照片）和毕业寄语卡片。

### 2. 样式设计 (Vanilla CSS)
- **配色**：
  - 北大红：`#9C2C2C` 作为仪式感主色。
  - 恋爱粉：`#FFF5F5` 作为背景，配有漂浮粉色心形气泡。
  - 胜利金：`#D4AF37` 用于胜利徽章。
- **卡片特效**：
  - `backdrop-filter: blur(12px)` 实现高端毛玻璃卡片质感。
  - 使用 CSS keyframe 制作气泡浮动、按钮呼吸和字效连击震动。
- **适配**：使用 CSS flexbox & grid，通过 `dvh`（Dynamic Viewport Height）保证在 iOS/Android 手机各种屏高下完美适配，不产生多余滚动条。

### 3. 素材与资源配置

  - `assets/photo_1.jpg`: 教室独照 (input_file_1.png)
  - `assets/photo_2.jpg`: 毕业典礼获证照 (input_file_2.png)
  - `assets/photo_3.jpg`: 与校长的合照 (input_file_3.png)

- 音效与BGM：
  - 使用 HTML5 Audio API，默认采用免费优质钢琴乐曲 URL 作为背景音乐。
